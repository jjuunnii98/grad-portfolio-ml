# grad-portfolio-ml

Graduate school portfolio repository focused on machine learning,
statistical modeling, and healthcare data analysis.
This repository documents my preparation process for graduate-level research.

머신러닝, 통계적 모델링, 헬스케어 데이터 분석을 중심으로 한 대학원 진학 준비용 포트폴리오 저장소입니다.
연구자로서의 사고 과정과 분석 역량을 체계적으로 기록하는 것을 목표로 합니다.

---

## Purpose of this repository

### English
The goal of this repository is not only to implement models,
but also to demonstrate research readiness:
- clear problem formulation
- understanding of assumptions
- reproducible analysis
- critical interpretation of results

This repository emphasizes *how* and *why* methods are used,
not just *what* is implemented.

### 한국어
이 저장소의 목적은 단순히 모델을 구현하는 것이 아니라,
다음과 같은 **연구 준비 역량**을 보여주는 데 있습니다.

- 문제 정의 능력
- 모델의 가정에 대한 이해
- 재현 가능한 분석 과정
- 결과에 대한 비판적 해석

“무엇을 구현했는가”보다  
**“왜 이 방법을 선택했고, 어떻게 해석했는가”**에 집중합니다.

## Focus areas

### 1. Survival analysis

#### English
- Kaplan–Meier estimator
- Cox proportional hazards model
- model diagnostics and interpretation
- extensions (planned): competing risks, time-dependent covariates

#### 한국어
- Kaplan–Meier 생존 함수 추정
- Cox 비례위험 모형
- 모델 진단 및 해석
- 확장 주제(예정): 경쟁위험 모형, 시간의존 공변량

---

### 2. Machine learning for tabular data

#### English
- baseline models (logistic regression, tree-based models)
- feature handling and preprocessing
- model evaluation and comparison

#### 한국어
- 기본 머신러닝 모델 (로지스틱 회귀, 트리 기반 모델)
- 변수 전처리 및 특성 처리
- 모델 성능 평가 및 비교

---

### 3. Paper replication

#### English
- replication of key methods from academic papers
- documentation of assumptions and limitations
- comparison between paper results and reproduced results

#### 한국어
- 주요 학술 논문의 방법론 재현
- 가정과 한계에 대한 명시적 정리
- 논문 결과와 재현 결과의 비교 분석

## Repository structure

### English
- `survival_analysis/`  
  Theory notes, simulations, and applied survival models

- `ml_baselines/`  
  Machine learning baselines for structured data

- `paper_replication/`  
  Paper summaries, implementations, and replication notes

- `notes/`  
  Study notes, interview preparation points, and research ideas

### 한국어
- `survival_analysis/`  
  생존분석 이론 정리, 시뮬레이션, 실제 데이터 적용

- `ml_baselines/`  
  구조화 데이터 기반 머신러닝 기본 모델 정리

- `paper_replication/`  
  논문 요약, 구현 코드, 재현 결과 정리

- `notes/`  
  연구 아이디어, 면접 대비 정리 노트

## Principles

### English
- Reproducibility over complexity
- Clear assumptions and limitations
- Simple baselines before complex models
- Documentation-first approach

### 한국어
- 복잡함보다 재현성을 우선
- 가정과 한계를 명확히 기술
- 복잡한 모델보다 기본 모델부터 검증
- 문서화를 우선하는 분석 방식
